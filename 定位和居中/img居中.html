1.水平居中：直接加text-align:center

2.垂直水平居中：

1..parent{line-height:400px;text-align:center;}
img{vertical-align:middle;}
重点：vertical-align只对非块状元素有效，如果要居中一个块状元素，那么就display：inline-block;

2.给img加一层div .tableCell
.parent{
display:table;
}
.tableCell{
display:table-cell;
vertical-align:middle;
}

3.img{
position:absolute;
top:0;
left:0;
right:0;
bottom:0;
margin:auto;
}

4.img{
position:absolute;
top:50%;
left:50%;
transform:translate(-50%,-50%);
}

5..parent{
display:flex;
flex-direction:row;
justify-content: center;//水平居中
align-items: center;//垂直居中
}
